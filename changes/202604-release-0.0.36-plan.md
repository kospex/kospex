## Summary of What's New in 0.0.36

This is a significant release with 4+ months of changes. Major highlights:

1. **Flask removed, FastAPI only** — completed migration, removed Flask/Werkzeug deps
2. **FastAPI/Starlette upgrade** — pinned FastAPI 0.135.3, Starlette 1.0.0, uvicorn 0.43.0; migrated all 47 TemplateResponse calls (#89)
3. **HabitatConfig** — centralized singleton for all kospex paths and configuration
4. **DuckDB ingestion MVP** — GitDuckDB and GitIngest for scalable git data storage
5. **Developer stats** — new developer_stats table for key person/bus factor analysis
6. **Assessment types** — standardized output filenames and assessments directory
7. **PyGitHub removed** — replaced with direct REST API calls (fewer dependencies)
8. **CLI renamed** — kospex.py -> kospex_cli.py to avoid package namespace clash
9. **Commit history** — new /history/ web endpoint
10. **Security fix** — updated vulnerable urllib3
11. **Python 3.12** — CI and Docker updated
